REPL Trailpack. Adds an interactive shell to your Trails app to help with
development and debugging.

## Usage
The repl trailpack is loaded in your trailpack config per usual.

```js
// config/trailpack.js
module.exports = {
  // ...
  packs: [
    // ...
    require('trailpack-repl')
  ]
}
```
